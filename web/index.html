Cydintosh Web Flasher

Flash the locally built Cydintosh firmware to an ESP32 Cheap Yellow Display board directly from Chrome or Edge, with no native install required on the client machine.

Use Chrome or Edge on desktop. Firefox and Safari do not support Web Serial here. This page must be served over https:// or from a trusted local origin.
This flashes firmware built from the current local workspace. It includes a merged ESP32 image plus the LittleFS image. Keep the board in bootloader/flashing mode before connecting if auto-reset does not work.

How to use

  1. Open this page in Chrome or Edge.
  2. Connect the CYD board over USB with a data-capable cable.
  3. If needed, put the board into flashing mode before clicking the button.
  4. Click Connect and Flash Cydintosh.
  5. Select the serial device presented by your OS.
  6. Choose Install and wait for flashing to complete.

Included images

  • merged-firmware.bin at offset 0x000000
  • littlefs.bin at offset 0x290000

The merged image contains the bootloader, partitions, application firmware, and patched Mac ROM.

Notes

  • Inspired by the same browser flashing pattern used by projects like TRMNL.
  • Uses ESP Web Tools, which wraps Espressif flashing flows via Web Serial.
  • If flashing fails, retry with the device manually held in boot mode and re-open the page.
